class Grammar

Reads a context-free grammar description, based on a given customisable syntax, and expands the grammar from a given root symbol.

constructor(symbols: MutableMap<String, Symbol>, modifiers: MutableMap<String, Modifier>, random: Random, syntax: GrammarSyntax)

Reads a context-free grammar description, based on a given customisable syntax, and expands the grammar from a given root symbol.

Parameters

Name Description
symbols: MutableMap<String, Symbol>

maps symbol names to their rules (Symbol), as described by the grammar
modifiers: MutableMap<String, Modifier>

maps modifier names (as used in the grammar definition) to the corresponding Modifier class describing their logic
random: Random

instance of Random used to pick rules when expanding symbols
syntax: GrammarSyntax

the syntax used to describe the grammar
